<h1 align="left">Steps to run this code</h1>

###

## Run Locally

Navigate to project directory and open terminal

Install dependencies

```bash
  pip install
```

Start the server

```bash
  flask run
```

## Tech Stack

**Client:** Html, CSS, Javascript

**Server:** Flask
